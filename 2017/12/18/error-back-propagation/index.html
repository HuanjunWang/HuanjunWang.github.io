误差逆传播算法BP

12月 18, 2017 发布在 Machine Learning

Error BackPropagation

误差拟传播算法，是最成功的神经网络学习算法。BP网络通常指用误差逆传播算法训练的前馈神经网络。
BP算法基于梯度下降策略，以目标的负梯度方向对参数进行调整。
